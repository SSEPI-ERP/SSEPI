Formatos aceptados

Excel (.xlsx, .xls) y CSV. Se detectan solas las hojas con encabezados (varias hojas por archivo). Los datos se actualizan si el SKU ya existe en la categoría seleccionada, o se importan como nuevos y se guardan en la base de datos (Supabase).

Ejemplos de columnas reconocidas (cualquier orden):

  • SKU / Código: Código Marking, SKU, Modelo, Clave, Ref
  • Nombre: Descripción del producto, Nombre, Producto
  • Stock: Existencia, Cantidad, Stock, Qty
  • Ubicación: Ubicación, Estante, Almacén
  • Costo: Costo unitario MXN, C/U, Costo
  • Precio: Precio venta, Total línea MXN, Precios SSEPI, Importe

Se busca automáticamente la fila de encabezados (incluso si hay títulos arriba como "INVENTARIO 2025").
