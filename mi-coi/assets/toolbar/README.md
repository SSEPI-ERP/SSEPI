# Iconos de toolbar/sidebar

Esta carpeta contiene iconos PNG opcionales para los botones laterales.

Si el archivo existe, la UI usa imagen; si no existe, cae automáticamente a emoji.

Nombres sugeridos usados por la app:

- `add.png`
- `edit.png`
- `del.png`
- `sync.png`

Puedes agregar más archivos y referenciarlos desde `frontend/main_window.py` mediante `asset_png="archivo.png"`.
